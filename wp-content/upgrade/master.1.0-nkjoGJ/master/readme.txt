"Master" is a free, personal blog design, with a very clean and elegant style. Along with the great design, we've also included all of the other mordern qualities in this theme to make your stay comfortable. This wordpress theme is fully compatible with Wordpress version 2.7


Features ::
---------------
1. Modern and beautiful web typeface.
2. Tableless design and 100% CSS-based layout.
3. 2 columns of fixed width.
4. Widget Ready.
5. XHTML 1.0 Transitional valid.
6. CSS 2.1 valid.
7. Search Engine optimized coding make wise use of h1, h3, h3.


Installation ::
---------------
STEP 1. Extract the files it contains. You need to preserve the directory structure in the archive when extracting these files.
STEP 2. Upload the theme to your /wp-content/themes/ directory.
STEP 3. Now go to Appearance >> Themes and activate the "Master" theme.

NOTE: If you still got confused then please visit http://codex.wordpress.org/Using_Themes for detailed installation instructions.
